# THEMED-TUG-OF-WAR
Kinetic Word Tug-of-War is a fast-paced 2-player duel! Unscramble letters to pull the rope. Type "ACT" or "CAT"—both count! Master 4 themes: Animals, Fruits, Home, and Colors. Features realistic rope physics, straining character animations, and a penalty system for typos. Outsmart your rival and pull them across the line to win!.
